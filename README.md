# Uma Linha a Cultura

Reviva 70 anos de música brasileira com arte e animação.

## Sobre o Projeto

Uma Linha a Cultura é um projeto que celebra 7 décadas da rica história musical brasileira através de arte e animação. Idealizado e criado por Sabrina Veiga, João Cincurá, Anderson Carvalho e Maria Cecília Souto.

## Tecnologias

- **Frontend**: React, TypeScript, Vite, TailwindCSS
- **Backend**: Node.js, Express
- **UI Components**: Radix UI, Shadcn/ui

## Como Executar

### Pré-requisitos

- Node.js 18+ 
- npm ou yarn

### Instalação

1. Clone o repositório:
```bash
git clone https://github.com/midafite/Uma-Linha-a-Cultura.git
cd Uma-Linha-a-Cultura
```

2. Instale as dependências:
```bash
npm install
```

3. Execute em modo de desenvolvimento:
```bash
npm run dev
```

4. Acesse `http://localhost:5000` no navegador.

### Build para Produção

```bash
npm run build
npm start
```
